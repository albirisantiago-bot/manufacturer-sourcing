1. Enter product type (e.g., "takeaway boxes", "cake boxes")

1. Click "Copy for Google Sheets" button
2. Open Google Sheets
3. Paste (Ctrl+V or Cmd+V)

### Column Structure
Results are formatted with these columns:
- `#` - Row number
- `Source` - Data source (Google, Web, etc.)
- `Company` - Manufacturer name
- `Comments` - Description and capabilities
- `Special Reference` - Notable features
- `Country` - Country code (TUR, PL, etc.)
- `Website` - Company website
- `Email` - Contact email
- `Phone` - Contact phone
- `Address` - Physical address

## ğŸ¨ Customization

### Adding New Manufacturers
Edit `app.js` and add to the `manufacturerDatabase` object:

```javascript
'CountryName': {
    'product type': [
        {
            company: 'Company Name',
            comments: 'Description',
            specialRef: 'Special features',
            website: 'https://...',
            email: 'email@...',
            phone: '+xx xxx xxx',
            address: 'Address'
        }
    ]
}
```

### Adding New Countries
1. Add country to select options in `index.html`
2. Add flag emoji to `countryFlags` object in `app.js`
3. Add country code to `countryCodes` object in `app.js`
4. Add manufacturers to `manufacturerDatabase` in `app.js`

### Styling
All styles are in `index.html` `<style>` section. Edit CSS variables in `:root`:

```css
:root {
    --primary: #0A4D3C;
    --accent: #F59E0B;
    --bg-dark: #0D1117;
    /* ... */
}
```
